# Soker-MovingHakerTools 移动应用项目开发文档

**版本：** 1.2  
**日期：** 2025年12月23日  
**开发者：** 吴永贤  
**项目状态：** 规划阶段（已更新）

---

## 1. 引言

### 1.1 编写目的

本文档详细描述移动应用 **Soker-MovingHakerTools** 的开发需求、功能设计、运行环境和技术约束，为后续开发、测试和维护提供统一指导。

本应用定位为 **轻量级移动工具集合**，所有功能 **不依赖开发者自建服务器和数据库（无后端数据存储）**。部分功能需联网调用第三方 API 或进行网络请求，以实现高级特性。

---

### 1.2 项目背景

- **项目名称：** Soker-MovingHakerTools  
- **开发者：** 吴永贤  

该应用面向移动端用户，提供多种实用工具，强调 **轻量、集成化、高效操作与隐私保护**。  
所有用户数据仅存储在本地设备中，不涉及云端同步或服务器存储。

---

### 1.3 术语说明

| 术语 | 说明 |
|----|----|
| Termux | Android 终端模拟器与 Linux 环境应用，支持运行 Python 等脚本 |
| 悬浮窗 | Android 系统允许应用窗口悬浮显示的功能 |
| 第三方 API | 公开的云服务接口（如身份证识别、AI 模型 API 等） |
| 无服务器依赖 | 不依赖开发者自建后端服务器和数据库 |

- Android 官方开发文档  
- Termux 开源项目文档

## 2. 项目概述

### 2.1 项目目标

开发一款 **轻量、实用、安全** 的移动工具应用：

- 不依赖自建服务器与数据库  
- 所有数据本地存储  
- 通过第三方公开 API 实现联网功能  
- 支持离线与在线功能共存

---

### 2.2 用户群体

- 技术爱好者、开发者  
- 需要集成化移动工具的用户  
- Android 设备用户（优先支持 Android 10+）

---

### 2.3 运行环境

- **平台：** Android  
- **最低版本：** Android 8.0  
- **推荐版本：** Android 12+  

**权限需求：**

- 相机（身份证识别）
- 存储读写
- 悬浮窗
- 网络与局域网访问

**其他要求：**

- 无自建服务器和数据库
- 应用体积 ≤ 50MB

---

### 2.4 约束与限制

- 所有用户数据仅本地存储  
- 联网功能必须明确提示隐私与数据上传行为  
- 严格遵守 Android 权限管理规范  
- 遵守第三方 API 使用协议  

---

## 3. 功能需求

### 3.1 总体功能列表

| 序号 | 功能模块 | 主要描述 | 是否需联网 |
|----|----|----|----|
| 1 | 身份证识别 | 独立模块，调用云 API 进行身份证识别 | 是 |
| 2 | 身份证识别登录 | 基于身份证识别结果进行本地身份验证 | 首次需联网 |
| 3 | 主页欢迎 | 功能入口与个性化欢迎页 | 否 |
| 4 | Termux Python 环境 | 集成 Termux，运行 Python 脚本 | 否 |
| 5 | 高级网页请求 | 自定义 Header / JS 注入 / 发包工具 | 是 |
| 6 | 局域网文件传输 | Wi-Fi 局域网文件共享与协作编辑 | 是（局域网） |
| 7 | 音乐播放 | 本地音乐播放器 | 否 |
| 8 | AI 接口 | 调用第三方 AI 云 API | 是 |
| 9 | 笔记功能 | Markdown / DOCX / XLSX 笔记 | 否 |
| 10 | 悬浮窗 | 工具悬浮显示 | 视模块而定 |
| 11 | 文件管理与预览 | 多格式本地文件预览 | 否 |

---

### 3.2 详细功能描述

#### 3.2.1 身份证识别（独立模块）

- 相机拍摄或相册导入身份证照片  
- 调用第三方身份证识别云 API  
- 返回结构化数据（姓名、证号、地址等）  
- 支持识别结果预览与手动修正  
- 本地缓存必要信息（哈希化存储）

---

#### 3.2.2 身份证识别登录

- 基于身份证识别模块  
- 本地保存身份信息哈希  
- 支持备用登录方式  

---

#### 3.2.3 主页欢迎

- 显示用户姓名  
- 功能入口网格布局  
- 支持主题与快捷方式自定义  

---

#### 3.2.4 接入 Termux Python 运行环境

- 需用户自行安装 Termux  
- 内嵌终端视图  
- 支持 Python 脚本编辑与运行  

---

#### 3.2.5 高级网页请求与预览

- 支持 GET / POST / PUT 等请求  
- 自定义 Headers、Cookies、代理  
- JS 注入、响应拦截  
- 内置网页预览与 HTML 保存  

---

#### 3.2.6 局域网文件传输（共享编辑）

- 自动扫描局域网设备  
- 文件发送与接收  
- 基于 WebSocket 的局域网协作编辑  

---

#### 3.2.7 音乐播放功能

- 扫描本地音乐  
- 后台播放  
- 通知栏控制  
- 简单均衡器  

---

#### 3.2.8 AI 接口

- 支持多个第三方 AI API  
- 聊天、文本生成、代码生成  
- 本地保存历史记录  

---

#### 3.2.9 笔记功能

- Markdown 实时预览  
- DOCX / XLSX 导入导出  
- 本地分类与搜索  

---

#### 3.2.10 悬浮窗功能

- 支持终端、AI、笔记、网页等模块  
- 可拖拽、缩放、透明度调节  

---

#### 3.2.11 多文件存储与预览

- 本地文件管理器  
- 支持图片、PDF、视频、办公文档预览  

---

## 4. 非功能需求

### 4.1 性能需求

- 启动时间 < 3 秒  
- 本地操作响应 < 1 秒  
- 联网功能目标响应 < 5 秒  

---

### 4.2 安全性需求

- 数据上传前明确隐私提示  
- 本地数据加密存储  
- API Key 本地加密保存  
- 权限最小化原则  

---

### 4.3 可用性需求

- 简洁 UI（Material Design）  
- 支持暗黑模式  
- 多语言（中文优先）  
- 离线降级提示  

---

### 4.4 可靠性需求

- 网络异常优雅处理  
- 完整异常捕获  
- 可选本地日志记录  

---

## 5. 外部接口需求

### 5.1 用户界面

- 主界面：底部导航栏 + 侧边抽屉  
- 联网模块独立入口  

---

### 5.2 软件接口

- Termux API  
- 第三方云 API（身份证识别、AI 模型等）  
- Android 系统 API（相机、存储、悬浮窗、网络）
